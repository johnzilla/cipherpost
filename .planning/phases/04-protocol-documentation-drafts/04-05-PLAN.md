<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Docs → readers | Broken anchors or missing cross-references reduce reader comprehension — link-check enforces integrity |
| lychee-action → CI build | The action runs with repo read access in CI; SHA-pin prevents supply-chain tag-swap attack |
| README → docs discoverability | Missing README link = docs exist but unreachable for a new reader |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-04-05-01 | Tampering | lychee-action replaced by malicious upstream tag | mitigate | SHA pin (`lycheeverse/lychee-action@<40-hex-chars>`) — tag-swap attacks don't affect SHA-pinned workflows; acceptance criterion enforces 40-char SHA format |
| T-04-05-02 | Denial of service | Anchor drift between SPEC and THREAT-MODEL silently breaks links | mitigate | Local lychee pass in Task 1 + Task 2 Step D before committing; CI job runs on every PR/push catching future drift |
| T-04-05-03 | Spoofing | `.lycheeignore` used to hide legitimate broken links rather than fix them | mitigate | Task 1 action explicitly says "Do NOT just add to `.lycheeignore`" for legitimate breakage; the escape hatch is only for documented slug-generation false positives or auth-gated URLs (the GitHub Advisory creation URL is the only pre-populated entry, justified by D-SEC-03 manual round-trip verification) |
| T-04-05-04 | Information disclosure | Status line in README.md exposes a stale "Phase 1 underway" when we're actually finishing Phase 4 | accept | Planner's call (per 04-CONTEXT.md Claude's Discretion): leave status untouched; a future `/gsd-transition` run updates README status at milestone close |
| T-04-05-05 | Repudiation | Phase claims to satisfy DOC-04 but one of the three docs is missing the cclink reference | mitigate | Task 2 Step E + Step G grep-verify `github.com/johnzilla/cclink` count `>= 1` in each of SPEC/THREAT-MODEL/SECURITY |
